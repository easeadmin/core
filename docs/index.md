# 什么是 EaseAdmin？
EaseAdmin 是一个基于 AdonisJS 和 Amis 构建的后台管理系统。
只需很少的代码即可构建出功能完善的 CRUD 页面，并且在 Amis 的加持下，无需写前端代码即可支持自定义页面。
基于 TypeScript 语言，采用经典 MVC 设计模式，丰富的 NPM 生态，真正做到 All in One 的全栈开发框架。

# 内置功能
* 账号管理
* 角色管理
* 权限管理
* 菜单管理
* 多语言管理
* 代码生成器
* amis 编辑器（待开发）
* helpers 开发脚手架（待开发）
