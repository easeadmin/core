# Introduction

EaseAdmin is the simplest and fastest way to build production-ready admin panels using `Typescript`.
Based on `Adonisjs` and `Amis` package, truly achieving full stack development

Built-in Features
* Account Management
* Role Management
* Permission Management
* Menu Management
* Multi-language Management
* Code Generator
* amis Editor (to be developed)
* Helpers Development Scaffold (to be developed)
